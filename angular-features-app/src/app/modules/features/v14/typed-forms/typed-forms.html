Angular 14: Typed Reactive Forms Strictly Typed FormGroup, FormControl & FormArray

Angular 14 introduced a powerful enhancement to reactive forms by supporting strict typing. You can now define the types of values inside your forms using FormGroup<T>, FormControl<T>, and FormArray<T>.

Main Benefits:

  • ✅ Improved TypeScript support and type inference
  • ✅ Better autocomplete and compile-time error detection
  • ✅ Use of nonNullable to prevent null values in required fields

Typed FormGroup Example:


export interface ContactForm {{ '{' }}
  name: FormControl<string>;
  email: FormControl<string>;
  message: FormControl<string | null>;
{{ '}' }}

this.contactForm = this.fb.group({{ '{' }}
  name: new FormControl('', {{ '{' }} nonNullable: true {{ '}' }}),
  email: new FormControl('', {{ '{' }} validators: [Validators.required, Validators.email], nonNullable: true {{ '}' }}),
  message: new FormControl('', {{ '{' }} nonNullable: false {{ '}' }})
{{ '}' }});

Typed FormArray Example:


export interface PhoneForm {{ '{' }}
  phones: FormArray<FormControl<number | null>>;
{{ '}' }}

this.phoneForm = this.fb.group({{ '{' }}
  phones: this.fb.array([
    this.fb.control<number | null>(null, Validators.required)
  ])
{{ '}' }});

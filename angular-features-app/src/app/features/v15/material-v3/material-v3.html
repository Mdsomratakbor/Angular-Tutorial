Angular 15 – MDC-Based Components & Router Standalone API

MDC-Based Components

Angular 15 has refactored many Material components to be based on Material Design Components for Web (MDC). This change improves accessibility for users with assistive technologies and ensures Angular Material components stay consistent with Google's official Material Design guidelines.

Key benefits include:

  • Enhanced accessibility out-of-the-box
  • Better alignment with Material Design standards
  • Easier future updates as Material Design evolves

Common components such as buttons, sliders, checkboxes, and dialogs have been updated to use MDC foundations and styling, resulting in a more consistent and accessible UI.
